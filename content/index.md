# 舞蹈训练体系
## 序言
很多舞者每天拉伸，但很少练力量和身体功能性，导致柔韧性越来越大，组织稳定越来越差。最终：

● 盂唇损伤
● 腰痛
● 髌股疼痛

就会出现。
真正保护舞者身体的，不是“更软”，而是在大活动度下，依然能稳定控制力量的能力。
真正高级的训练思路不是：“练更多动作”
而是建立能力
模块对舞者、运动员、康复训练来说，

# 伤病篇
舞蹈伤病的核心逻辑不是“随机发生”，而是：

> **控制系统失衡 → 结构过载 → 局部受伤**

## 三个最核心问题来源：

### ① 足踝不稳

→ 影响全身链条

### ② 髋控制不足

→ 导致膝和腰代偿

### ③ 核心稳定差

→ 导致整体“散掉”

> 舞蹈伤病不是“某个部位坏了”，而是“控制系统失衡后，某个部位先撑不住”。
