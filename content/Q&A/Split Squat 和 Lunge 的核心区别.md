# 先看最本质区别

| 动作   | [[分腿蹲（Split Squat）]] | [[弓步蹲（Lunge）]] |
| ---- | -------------------- | -------------- |
| 双脚   | 固定不动                 | 会移动            |
| 类型   | 稳定型                  | 动态型            |
| 重点   | 控制与稳定                | 减速与转换          |
| 难点   | 单腿稳定                 | 动态接受冲击         |
| 更像   | “力量底座”               | “运动表现”         |
| 难度   | 更基础（控制）              | 更高（协调）         |
| 舞蹈意义 | 单腿支撑                 | 步伐/跳跃连接        |

# 一、Split Squat 的核心目的

## 关键词：

“建立稳定”

## 它主要训练：

### 1. 单腿稳定（Single-leg stability）

这是核心。

因为：

● 现实运动中很多动作都是：一条腿承重

● 另一条腿移动

比如：

● 舞蹈arabesque

● 跑步支撑期

● 足球变向

● 跳跃落地

## 2. 骨盆控制（Pelvic control）

尤其训练：

### 臀中肌（Glute med）

作用：

● 防止：骨盆掉落

● 膝内扣

● 髋旋转

## 3. 力量输出基础

它更容易：

● 慢速控制

● 渐进负荷

● 精细调整动作

所以：

## 康复特别爱用它。

## 它像什么？

### “静态单腿深蹲”

核心是：

在稳定里发力。

# 二、Lunge 的核心目的

## 关键词：

“动态控制”

## 它训练什么？

### 1. 减速能力（Deceleration）

当你迈步出去时：

身体必须：

● 接受冲击

● 刹车

● 稳定重心

这非常接近真实运动。

## 2. 重心转换（Weight transfer）

运动里：

很少有人站着不动。

Lunge训练：

● 前后移动

● 重心切换

● 动态平衡

## 3. 动态协调

需要：

● 节奏

● 步态控制

● 下肢协同

## 它像什么？

### “受控的跌倒”

因为：

● 你其实是在：往前“掉”

● 再把自己拉回来

# 生物力学区别（重点）

# Split Squat

## 身体重心：

相对固定。

因此：

### 更强调：

● 局部肌肉张力

● 稳定控制

● 持续发力

# Lunge

## 身体重心：

持续移动。

因此：

### 更强调：

● 动态减速

● 冲击吸收

● 协调反应

# 为什么康复更常先练 Split Squat？

因为：

## 稳定先于动态。

如果一个人：

● 骨盆都控制不住

● 膝还在内扣

直接做 lunge：

很容易代偿。

# 经典顺序

## Rehab progression：

### Step 1

↓

### Step 2

Static lunge control

↓

### Step 3

Forward/reverse lunge

↓

### Step 4

跳跃/变向

# 对舞者的意义区别

# Split Squat 更像：

## “支撑腿能力”

例如：

● arabesque

● développé

● passé balance

核心：

单腿稳定。

# Lunge 更像：

## “运动转换能力”

例如：

● 跳跃落地

● traveling

● 动态转换

● 滑步

核心：

动态控制。

# 肌肉感受区别

# Split Squat

通常：

● 臀感更强

● 稳定肌参与更多

● 更容易慢练

# Lunge

通常：

● 心肺更累

● 协调需求更高

● 更接近功能运动

# 一个非常重要的区别

## Split Squat：

你可以“提前稳定”。

## Lunge：

你必须“边移动边稳定”。

这就是为什么：

Lunge 对神经系统要求更高。

# 训练中的实际选择

|   |   |
|---|---|
|目标|更适合|
|康复初期|Split squat|
|髋稳定|Split squat|
|臀中肌训练|Split squat|
|舞者单腿控制|Split squat|
|跑步专项|Lunge|
|变向能力|Lunge|
|动态减速|Lunge|
|功能性训练|Lunge|

# 一句话总结

## Split Squat

“先学会稳定地发力。”

## Lunge

“再学会在移动中控制力量。”

进退阶

Level 1

支撑式 split squat

扶墙/扶杆。

适合：

● 术后

● 疼痛期

Level 2

自重 split squat

重点：

● 控制

Level 3

Goblet split squat

抱哑铃增加负荷。

Level 4

Bulgarian split squat

后脚抬高：

难度明显提升：

● 髋稳定

● 平衡

● 臀肌负荷

都会增加。
