 ## 下腰和麦肯基疗法的区别

很多人第一次学脊柱生物力学时都会困惑：“既然前屈容易腰突，为什么后伸又会伤腰？

麦肯基（McKenzie）不是还让人做后伸吗？”关键在于：

“不同方向” ≠ “绝对安全或绝对危险” 真正决定结果的是：

● 哪个组织受力

● 有没有反复加载

● 是否已经受伤

● 力量如何分布

● 动作是否代偿

# 一、为什么“屈曲负重”最容易诱发典型腰突

经典腰突（尤其后外侧突出）最常见机制：

腰椎屈曲 + 负重 + 反复 因为：

当前屈时：

● 椎间盘前方被压缩

● 髓核倾向向后移动

● 后侧纤维环张力增加

如果再加：

● 搬重

● 扭转

● 久坐

后侧纤维环更容易疲劳。

所以很多人的腰突发生于：

● 搬箱子

● 弯腰拉东西

● 久坐后突然起身

# 二、那后伸为什么也可能伤腰？

因为：

后伸伤的通常不是“同一种结构” 后伸（尤其极端后弯）时：

### 椎间盘前部受拉

### 后方结构受压

包括：

● 小关节（facet joints）

● 椎弓

● 后柱结构

所以舞蹈/体操常见的是：

● 后伸痛

● 小关节卡压

● [[椎弓峡部裂（Lumbar Spondylolysis）|椎弓峡部裂]]

而不是典型“搬重物型腰突”。

# 三、所以前屈和后伸都可能出问题

只是：

|   |   |
|---|---|
|动作|更容易受伤结构|
|屈曲负重|椎间盘后侧纤维环|
|极端后伸|小关节、椎弓、后柱|
|后伸+旋转|椎弓应力损伤风险更高|

所以：

脊柱没有“万能安全方向” 只有：

● 当前状态是否适合

● 是否超负荷

● 是否长期重复

# 四、那为什么麦肯基用后伸？

因为：

## 麦肯基不是“练腰后弯”

它本质是：

方向偏好（Directional Preference） 很多腰突患者有一种现象：

### 中央化（centralization）

比如：

● 本来腿痛

● 做几次后伸后

● 腿痛减轻

● 痛回到腰中央

这说明：

某些人的椎间盘症状对后伸反应好。

## 麦肯基后伸的目的

通常是：

### 减少神经根刺激

### 改变椎间盘压力分布

### 暂时缓解后方突出刺激

它不是：

● 把腰拼命折断

● 做舞蹈式大下腰

● 长时间压极限后弯

而是：

● 小幅重复

● 可控

● 无症状恶化

的治疗性动作。

# 五、麦肯基并不适合所有人

有些人：

● 后伸更痛

● 有明显小关节问题

● 椎弓峡部裂

● 伸展不耐受

那后伸可能反而加重。

所以真正的麦肯基评估核心是：

哪个方向让症状“中央化” 而不是：

“所有腰痛都做后伸”

# 六、舞蹈生最容易混淆的点

很多舞蹈生会说：

“我天天后弯，为什么还腰突？”

因为舞蹈中的问题常常是：

极端后伸 + 剪切力 + 旋转 + 疲劳代偿 这和：

治疗性麦肯基后伸 完全不是一回事。

差别非常大：

|   |   |
|---|---|
|麦肯基后伸|舞蹈压腰|
|小幅度|极限活动度|
|可控|常代偿|
|低负荷|高频高负荷|
|观察症状变化|追求软度|
|治疗目的|表演目的|

# 七、真正重要的是“脊柱耐受方向”
### [[为什么脊柱耐受方向因人而异]]

有的人：

● 怕屈曲（flexion intolerant）

● 有的人怕后伸（extension intolerant）

比如：

### 屈曲不耐受

常见：

● 久坐痛

● 弯腰痛

● 晨起弯腰困难

### 后伸不耐受

常见：

● 后仰卡压

● 下腰尖锐痛

● 站久痛

舞蹈生很常见后者。

所以你可以把它理解成：

麦肯基后伸 ≠ “后伸永远安全” 屈曲伤腰 ≠ “后伸完全无害” 真正决定结果的是：

● 组织耐受

● 动作质量

● 训练剂量

● 是否代偿

● 是否已经存在损伤。

脊柱耐受方向（directional tolerance / directional preference）非常因人而异。

这其实是现代脊柱康复里很核心的概念。同一个动作：

● 对一个人是“缓解”

● 对另一个人可能是“刺激”
